This program is meant to implement a Last-to-Start, First-to-Finish activity selection algorithm.
This means the program is supposed to select data pieces to start as late as possible and end as quickly as possible.
The end result will return the number of activities selected as well as which activities were chosen.
This folder contains the following:
activitySelection.py

To Compile and Run:
chmod +x activitySelection.py
./activitySelection.py

OR

python3 activitySelection.py
